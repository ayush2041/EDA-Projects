# 📊 Exploratory Data Analysis (EDA) Projects

A collection of Exploratory Data Analysis (EDA) projects showcasing data cleaning, preprocessing, visualization, statistical analysis, and business insights using Python.

## 🚀 About

This repository contains multiple real-world EDA projects completed as part of my Data Science & AI/ML learning journey. Each project focuses on understanding datasets, cleaning data, identifying patterns, and extracting meaningful insights through visualization.

## 🛠️ Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

# 📂 Projects

## 1️⃣ Customer Churn Analysis

### Objective
Analyze customer behavior to identify factors affecting customer churn.

### Key Tasks
- Data Cleaning
- Missing Value Treatment
- Feature Analysis
- Correlation Analysis
- Customer Segmentation
- Data Visualization

### Insights
- Identified major reasons behind customer churn.
- Found important customer attributes influencing retention.
- Suggested business strategies to reduce churn.
